## Installation

```bash
npm install @pantheon-ai/opencode-font
```

## Quick Start

### Blocky Pixel-Art SVG (Recommended for Logos)

Perfect for creating retro-style logos and headers with theme support:

```typescript
import { blockyTextToSVG } from '@pantheon-ai/opencode-font';

const svg = blockyTextToSVG('OPENCODE', {
  theme: 'dark', // or 'light'
  blockSize: 6, // Size of each pixel block
  charSpacing: 1, // Space between characters
  optimize: true, // Reduces file size by 30-40%
});

// Inject into your page
document.getElementById('logo').innerHTML = svg;
```

### Simple Font-Based SVG

Great for dynamic text rendering with custom styling:

```typescript
import { convertTextToSVG } from '@pantheon-ai/opencode-font';

const svg = convertTextToSVG('HELLO', {
  fontSize: 48,
  color: '#00ff00',
});

document.getElementById('text').innerHTML = svg;
```

### CSS Font Integration

Use the font directly in your stylesheets:

```html
<link rel="stylesheet" href="node_modules/@pantheon-ai/opencode-font/css/opencode-font.css" />

<style>
  .pixel-text {
    font-family: 'OpenCodeLogo', monospace;
    font-size: 48px;
    color: #00ff00;
    text-transform: uppercase;
  }
</style>

<h1 class="pixel-text">OPENCODE</h1>
```

## Character Support

The font includes **32 glyphs**:

- **Letters**: A-Z (uppercase only)
- **Symbols**: `-` (hyphen), `|` (pipe), `'` (apostrophe), `"` (quote), `?` (question), `!` (exclamation)

Check out the [Glyph Showcase](/opencode-font/glyphs/) to see all available characters rendered as SVG.

## API Comparison

| Feature         | `blockyTextToSVG()` | `convertTextToSVG()` |
| --------------- | ------------------- | -------------------- |
| Theme Support   | ✅ Light/Dark       | ❌ Single color      |
| File Size       | Small (optimized)   | Very small           |
| Customization   | High                | Medium               |
| Best For        | Logos, headers      | Dynamic text         |
| Browser Support | All modern browsers | All modern browsers  |
